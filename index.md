# System Administration and Management Documentation

This series of reports documents system administration and management using Windows Server 2025 and a Windows workstation. The reports cover the installation/configuration of server features, the automation of creating 2000+ users, the creation and implementation of group policies for user groups, the isolated DHCP setup between the server and workstation, and the creation of backups for the server. These reports were made for assignments in a system administration and management class. Everything in the reports is done for a mock company called Lopes Manufacturing. Lopes Manufacturing's mission is to design, manufacture, assemble, and install high quality helicopter parts to ensure that U.S. military equipment performs flawlessly and allows personnel to safely execute missions in extreme environments and conditions.
